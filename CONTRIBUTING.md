# Contributing

Contributions are welcome! Here's how:

1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Make your changes
4. Open a pull request

## Ideas for contributions
- Windows Task Scheduler alternative to the launchd plist
- Linux cron setup guide
- Support for additional file types (PDF, YouTube transcripts, Twitter threads)
- Better local model prompts for higher quality wiki entries
- `wiki/synthesis/` auto-generation logic
- Auto-update of `wiki/index.md` after each ingest run
